BACKBONE.js
==============================

Getting Started
------------------------------

Dependencies: jquery, JSON2.js and underscore.js

Why backbone? Organize and handle a large amount of javascript code for a web application.

Architecture
------------------------------

The architecture of Backbone.js contains following modules:

- HTTP Request
vRouter
- View
- Events
- Model
- Collection
- Data Source
